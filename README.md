## Sobre el proyecto

Pokopia API es el backend del proyecto Pokopia, una API REST que provee informacion sobre Pokemon disponibles en el juego. Permite consultar datos como tipos, especialidades, habitats, climas, zonas de aparicion y evoluciones.

## Stack

- **Runtime**: Node.js 22
- **Framework**: Fastify 5
- **Lenguaje**: TypeScript
- **Validacion**: Zod
- **Imagenes**: Cloudflare R2
